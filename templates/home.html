Know limitations :

  • Youtube results (api & browser) can only provide ~500 results (to get more it need to be outdated every 500 results. see : how-can-get-all-results-from-youtube-api-search-api-response )
  • Automatic captions seems to cannot be retrieve via API. The author must havec activated CC on his video. It need to get a little tricky with xml version like : http://video.google.com/timedtext?lang=en&v=hSnB7zGW15M For more see : get-closed-caption-cc-for-youtube-video or : extract-automatic-captions-from-youtube-video
  • Comments get only one hierachical level
  • Search provide playlist incorporated in results as a video
